## 算法加速清单 ⚡⚡⚡

### 发布前优化
- [ ] 问题简单明确（降低回复门槛）
- [ ] 提供多个参与方式（评论/投票/转发）
- [ ] 添加视觉元素吸引注意
- [ ] 使用emoji增强可读性
- [ ] 设置明确的CTA

### 互动设计
- [ ] 问题有多个合理答案（避免单一正确答案）
- [ ] 话题相关性强（受众真正关心）
- [ ] 个性化空间大（让每个人都能贡献独特观点）
- [ ] 避免yes/no问题（追求开放式讨论）

### 发布后策略（前3小时至关重要！）
- [ ] 0-15分钟: 回复前5条评论
- [ ] 15-30分钟: 点赞所有新评论
- [ ] 30-60分钟: 深度回复高质量评论
- [ ] 1-3小时: 持续互动，创造讨论氛围
- [ ] 如获高互动(>50回复): 发推文串扩展讨论

## Grok AI 互动信号 ⭐⭐⭐

Grok AI 特别重视以下互动信号（按权重排序）:

1. **回复数** (最高权重)
   - 评论=深度兴趣
   - 算法会优先推送引发讨论的内容

2. **回复质量**
   - 长回复 > 短回复
   - 有意义的讨论 > 单字回复

3. **互动速度**
   - 前30分钟的互动最关键
   - 快速获得回复=内容质量高

4. **作者参与度**
   - 你回复评论的速度和质量
   - 高作者参与=高算法推荐

5. **转发评论比**
   - 纯转发<有评论的转发
   - 鼓励用户加评论转发

## 高级技巧 🎓

### 串联策略
在评论区自己发布补充信息或追问:
- "忘了说，[补充信息]"
- "另一个角度是..."
- 激发更多回复

### 时间炸弹
设置延迟公布结果:
- "1小时后公布答案"
- "明天同一时间揭晓"
- 让用户设提醒回来

### 话题标签战术
- 主推文: 1-2个核心标签
- 评论区: 添加更多相关标签
- 避免标签过载

### 视觉增强
- 投票: 使用emoji选项
- 对比: 用图表/截图
- 结果: 分享数据可视化
