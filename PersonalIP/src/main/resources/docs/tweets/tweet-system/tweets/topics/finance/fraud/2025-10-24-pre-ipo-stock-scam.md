# 原始股骗局:上市暴富梦的破灭

## 推文内容

"某某公司即将上市,现在买原始股,上市翻100倍!"

一个老板相信了这句话,投入2000万,最后血本无归。

**案例:虚假原始股诈骗案(2022年典型案例)**

**受害者:王总,建筑公司老板,52岁**

**诈骗全流程:**

**第1步:接触目标**
- 某"投资顾问"在高尔夫球场搭讪
- 自称某知名投资公司合伙人
- 递上精美名片,微信朋友圈都是成功人士合影

**第2步:建立信任**
连续3个月的"友情投资":
- 推荐的两只A股涨停(实为撒网式推荐)
- 邀请参加"投资峰会"(豪华酒店,请托儿烘托气氛)
- 介绍认识"上市公司高管"(实为同伙)

王总逐渐相信:"这是真正的高端圈子"

**第3步:抛出诱饵**

"顾问"神秘透露:
"有个绝密项目,某科技公司准备美股上市"
"现在内部募集最后一轮原始股"
"上市后至少翻50倍"

配套材料:
✅ 精美的商业计划书
✅ "律师事务所"出具的法律意见书
✅ "投资协议"加盖公章
✅ 公司参观(租用的豪华办公室)
✅ 技术演示(买来的产品原型)

**第4步:制造紧迫感**

"这轮只募集5000万,已经认购了4200万"
"名额有限,只给信得过的朋友"
"错过这次,永远没有机会"

王总动心了。

**第5步:逐步加码**

第1笔:200万试水
- 签订"股权认购协议"
- 获得精美的"股权证书"
- 承诺1年后上市

第2笔:800万加仓
- "顾问":"下个月就要关闭认购"
- 王总担心错过机会,追加投入

第3笔:1000万梭哈
- 抵押厂房贷款
- 凑齐最后一笔钱投入

总计:2000万元

**第6步:拖延跑路**

3个月后:
王总:"什么时候上市?"
顾问:"美国SEC审核延迟了"

6个月后:
王总:"为什么联系不上公司?"
顾问:"总部搬迁,新地址稍后告知"

1年后:
- 顾问失联
- 公司人去楼空
- "股权证书"是假的
- 工商查无此公司

**原始股骗局的8个套路:**

🎪 **虚构上市公司**
- 公司名字听起来很高大上
- 实际工商注册信息虚假或空壳

🎪 **伪造上市文件**
- 假的"上市批文"
- 假的"律师意见书"
- 假的"审计报告"

🎪 **虚假背景包装**
- 声称与政府机构合作
- 号称国资背景或外资注资
- 虚构国际合作项目

🎪 **制造稀缺性**
- "内部名额,不对外公开"
- "只给特定圈层的人"
- "马上截止认购"

🎪 **承诺天价回报**
- "上市翻100倍"
- "一年回本,五年财富自由"
- 画饼而不谈风险

🎪 **打造高端形象**
- 豪华办公室(租来的)
- 高端商务活动(骗子聚会)
- 合影名人(P图或蹭拍)

🎪 **层层推荐机制**
- 拉人头有返点
- 受害者变成加害者
- 传销式扩散

🎪 **拖延术**
- "审批流程需要时间"
- "市场环境不好延迟上市"
- 拖到受害者失去耐心

**真实数据:**
- 2020-2023年,全国破获原始股骗局500+起
- 涉案金额超过300亿元
- 受害者多为50岁以上有积蓄的企业主

**如何识别真假原始股?**

✅ **查询工商信息**
- 国家企业信用信息公示系统
- 查公司成立时间、注册资本、股东信息

✅ **验证上市进度**
- 真实上市公司会在证监会官网公示
- 美股上市在SEC官网可查
- 港股上市在香港交易所公示

✅ **核实投资协议**
- 正规股权投资有规范的投资协议
- 必须由正规律所见证
- 工商局备案可查

✅ **警惕这些话术**
- "保证上市"(没人能保证)
- "翻倍回报"(违反证券法)
- "内部消息"(涉嫌内幕交易)

✅ **了解投资门槛**
- 真实原始股投资门槛极高
- 需要合格投资者资格认证
- 普通人很难接触到真实项目

**防骗铁律:**

1️⃣ 不熟悉的投资渠道=骗局
2️⃣ 承诺高回报无风险=骗局
3️⃣ 制造紧迫感催促决策=骗局
4️⃣ 无法在官方渠道核实=骗局

**记住:**
真正的优质原始股,轮不到陌生人给你介绍。

能找上门来主动推销的"原始股",100%是骗局。
