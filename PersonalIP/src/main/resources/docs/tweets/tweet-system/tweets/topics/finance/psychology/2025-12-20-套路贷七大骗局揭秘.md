我的大学同学小张，去年因为"套路贷"

## 套路贷的7大常见骗局

### 骗局1：砍头息（虚增债务）

**小张的遭遇：**
```
说好借款：50,000元
实际到账：38,500元
被扣费用：
- "平台服务费"：3,000元（6%）
- "风险保证金"：5,000元（10%）
- "GPS安装费"：1,500元（3%）
- "征信查询费"：2,000元（4%）

但借条上写的是：50,000元！
```

**识破方法：**
✅ 要求对方提供完整费用明细
✅ 借条金额 = 实际到账金额
✅ 录音/录像保留证据
❌ 任何"先扣费用"的都要警惕

**真实数据：**
- 2023年全国套路贷案件：8,742起
- 平均虚增债务比例：2.3倍
- 受害者平均年龄：28岁

---

### 骗局2：阴阳合同（一式两份）

**操作手法：**
```
合同A（给你看的）：
借款50,000元，月息1.5%，期限12个月

合同B（实际生效的）：
借款50,000元，月息8%，违约金日息3%
逾期一天罚款5,000元
```

**小张的教训：**
签了"空白合同"，对方后期填写
结果月息从1.5%变成8%（年化96%！）

**防范措施：**
✅ 所有合同自己保留原件
✅ 拍照留存每一页（包括空白处）
✅ 绝不签空白合同
✅ 要求当场完整填写

**法律依据：**
《民法典》第680条：禁止高利放贷
年利率超过36%的部分无效

---

### 骗局3：软暴力催收

**小张经历的催收升级：**

**第1周（逾期1天）：**
- 电话轰炸：一天120个电话
- 短信攻击："老赖"、"诈骗犯"
- 微信骚扰：凌晨3点发威胁信息

**第2周（逾期7天）：**
- P图造谣：把小张P到通缉令上
- 爆通讯录：给所有联系人发短信
- 内容："欠债20万，玩失踪"

**第3周（逾期14天）：**
- 上门骚扰：半夜敲门
- 贴大字报：在小区贴"欠债不还"
- 威胁家人：打电话给父母单位

**第4周（逾期30天）：**
- 伪造律师函
- 假冒公安人员
- 威胁"抓人坐牢"

**应对方法：**
🚨 立即报警（保留所有证据）
📱 录音录像所有接触
📋 整理催收时间表
⚖️ 申请反骚扰保护令

**真实判例：**
2024年浙江某催收公司
非法拘禁罪：主犯判刑7年
涉案金额：1,200万元

### 骗局4：肆意认定违约

**设置陷阱：**
```
还款方式：每月1日12:00前转账到指定账户

实际操作：
- 还款日当天不接电话
- 银行账户"系统维护"
- 支付宝转账"未到账"
- 12:01就认定违约

违约后果：
违约金：50,000元 × 30% = 15,000元
一天逾期，债务变65,000元
```

**小张的坑：**
约定10月1日还款
对方10月1日"失联"
10月2日说已逾期，要付15,000违约金

**防范：**
✅ 提前3天还款（留证据）
✅ 银行转账（有记录）
✅ 保留转账截图 + 短信通知
✅ 录屏还款全过程

### 骗局5：转单平账（雪球越滚越大）

**债务是怎么从5万变成20万的：**

```
时间轴：

2024年1月：
借款A平台：50,000元
月息1.5%，实际年化18%（还算正常）

2024年2月（第一次还不上）：
被介绍到B平台"转单"
借70,000元还A平台的50,000元
剩余20,000到手，还各种费用

2024年3月：
B平台要还款了，又被介绍到C平台
借90,000元还B平台的70,000元
剩余20,000，又是各种费用

2024年4月：
C平台到期，被介绍到D平台（高利贷团伙）
借120,000元，月息8%

就这样，3个月欠款从5万变成12万
```

**警惕话术：**
❌ "我帮你找个平台，利息更低"
❌ "以贷养贷，先周转一下"
❌ "这个平台审批快，马上到账"
❌ "我们是战略合作伙伴"

**真相：**
他们是"一条龙"团伙
从A到D都是同一个幕后老板
目的就是让你债务越滚越大

### 骗局7："套路贷"伪装成正规平台

**迷惑性极强的特征：**

```
小张借款的"XX金融"表面看很正规：

✓ APP界面专业（其实是模板）
✓ 客服24小时在线（实际是话术团队）
✓ 有"营业执照"（假的或套牌）
✓ 声称"银监会批准"（假的）
✓ 有"用户协议"（霸王条款）
```

**如何识别正规平台：**

**查询渠道：**
1. 国家企业信用信息公示系统
   网址：www.gsxt.gov.cn
   查公司注册信息、经营范围

2. 中国人民银行征信中心
   网址：www.pbccrc.org.cn
   查是否有放贷资质

3. 银保监会官网
   查是否有金融牌照

**正规平台特征：**
✅ 有金融牌照（消费金融牌照/小贷牌照）
✅ 年化利率 ≤ 24%
✅ 费用明细清晰
✅ 合同规范（不签空白合同）
✅ 正规催收（不爆通讯录）

**知名正规平台举例：**
- 蚂蚁借呗（年化7.2%-18.25%）
- 微粒贷（年化7.2%-18%）
- 京东金条（年化9%-18%）
- 度小满（年化7.2%-24%）
