# 我是如何在一天之内亏掉3个月利润的

## 推文内容

昨天我犯了一个交易者最不该犯的错误：
破坏了自己的交易纪律 💔

事情是这样的：

上午10点，做空黄金，止损设在2330
价格到了2328，我心想"再给它一点空间"
把止损移到2335...

结果？直接拉到2342被扫损

正常情况下，亏损200刀我会停手
但昨天我想"扳回来"

于是：
第2单：-150刀
第3单：-220刀
第4单：-180刀

4小时内，3个月的利润全没了

冷静下来后，我复盘了整个过程：

**错误1：移动止损**
止损是保命的，不是拿来调整的

**错误2：报复性交易**
市场不欠你，亏了就是亏了

**错误3：偏离系统**
有交易计划却不执行，等于没有

**错误4：情绪化**
急着"扳回来"的心态是毒药

这次教训花了3个月的利润，但学到的东西可能价值更大：

交易纪律 > 交易技巧

现在我给自己立了新规矩：
1. 止损绝不移动
2. 单日亏损达到资金的2%，当天停止交易
3. 连续3单止损，暂停交易1天
4. 每笔交易都记录，周末复盘

如果你也做交易，请记住：
保住本金比赚钱更重要

你有过类似的经历吗？
