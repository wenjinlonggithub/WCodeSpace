**第1条评论（发布后立即）**:

我的答案是 C) 更好的特征工程

为什么？

基于我过去3年15个AI生产项目的数据：

📊 性能提升来源：
• 特征工程: 平均+42% F1提升
• 更多数据: 平均+28% F1提升
• 复杂模型: 平均+15% F1提升
• 更长训练: 平均+8% F1提升

**第2条评论（30分钟后，总结讨论）**:

看到大家的讨论很精彩！👏

几个关键洞察：

1. 答案其实是"看情况"（很多人提到）
2. 数据质量 > 数据数量
3. 好的特征 = 给模型正确的提示
4. 过度复杂的模型容易过拟合

最大的教训：
先用简单模型 + 好特征建立baseline
然后再考虑复杂化
