## 校园贷的7大常见诈骗手法

### 手法1："培训贷"

```
诈骗话术：
"同学，你想学Python吗？"
"我们是XX科技，和你们学校合作的"
"培训费12,000元，但可以分期"
"学完保证月薪15K+"

实际情况：
- 培训质量极差（网上免费课程）
- 推荐就业是假的
- 12,000元的贷款是真的
- 培训机构拿了钱就跑路
```

**真实案例（2023年北京）：**
某"AI培训机构"诈骗1,200名大学生
涉案金额：1,800万元
培训内容：全是网上免费教程

### 手法2："美容贷"

```
诈骗话术：
"小姐姐，你的皮肤底子很好"
"只需要做个小气泡+光子嫩肤"
"原价8,000，现在活动价3,999"
"可以分12期，每月只要333元"

实际情况：
- 美容效果几乎没有
- 成本不到500元
- 3,999元全是贷款
- 不做就不让走（软禁）
```

**真实案例（2024年杭州）：**
某"医美工作室"
诱导42名女大学生"美容贷"
平均每人负债2.8万元
有人做完后严重过敏，还要还贷款

### 手法3："兼职刷单贷"

```
诈骗话术：
"招聘兼职刷单员"
"日赚300-500元"
"任务简单，在家就能做"
"需要垫付，完成任务后返还本金+佣金"

实际操作：
第1单：垫付200元，返还220元 ✅
第2单：垫付500元，返还550元 ✅
第3单：垫付2,000元，拉黑 ❌

然后"客服"说：
"你的任务卡住了，需要继续垫付5,000元才能解冻"

学生没钱，"客服"说：
"我帮你申请贷款垫付吧"

就这样掉进了坑
```

**数据：**
2023年全国"刷单贷"受害者：8,900+人
平均损失：1.2万元/人

### 手法4："租房贷"（租金贷）

```
诈骗话术：
"同学,这个房子月租2,000元"
"但你可以用'分期支付'，每月只付1,500元"
"相当于打75折！"

实际情况：
- "分期支付"是贷款
- 你一次性贷款了24,000元（12个月房租）
- 钱给了租房平台
- 平台跑路，房东没收到钱
- 房东赶你走
- 你还要继续还贷款

你：无家可归 + 背债24,000元
```

**真实案例（2023年上海）：**
某"青年公寓"平台爆雷
影响租客：3,400人
大部分是应届毕业生
损失金额：超过5,000万元

### 手法5："注销校园贷"诈骗

```
诈骗话术：
"你好，我是XX银行客服"
"我们查到你在大学期间注册过校园贷账户"
"根据国家新规定，必须注销"

客服："你需要把你所有网贷平台的额度都借出来"

"验证完就可以注销，钱会原路返还"
```

## 如何识别校园贷/裸贷陷阱

### 🚨 危险信号（看到这些立刻警惕）

#### 1. 营销话术类
❌ "大学生专属"、"学生福利"
❌ "无需抵押"、"不查征信"
❌ "当天到账"、"秒批"
❌ "利息超低"、"0利息"（都是骗人的）
❌ "毕业后再还"
❌ "兼职就能还清"

#### 2. 索要信息类
❌ 要求提供裸照、不雅照片
❌ 要求提供所有社交账号密码
❌ 要求提供父母详细信息
❌ 要求下载不明APP
❌ 要求开通"免密支付"

#### 3. 合同陷阱类
❌ 签空白合同
❌ 电子合同不给复印/保存
❌ 合同页数很多（超过30页）
❌ 字体特别小（小于5号字）
❌ 催你"快点签，名额有限"

#### 4. 费用陷阱类
❌ 扣"砍头息"（到账金额<借款金额）
❌ 各种莫名其妙的费用：
   - 平台服务费
   - 征信查询费
   - 账户管理费
   - GPS安装费（你又不借车）
   - 风险保证金
❌ 利息不明确，只说"月息0.99%"

#### 5. 催收行为类
❌ 威胁公开裸照
❌ 爆通讯录
❌ 骚扰家人、同学
❌ 上门威胁
❌ 假冒公检法（"你涉嫌犯罪"）

**第1步：报警（最高优先级）**
```
拨打110或到派出所
说明："我遭遇校园贷诈骗和敲诈勒索"

带上：
- 手机（聊天记录）
- 身份证
- 学生证
- 借款合同（如果有）
```

**第2步：告诉家人（必须做）**
我知道很难开口，但：
- 你是受害者，不是罪犯
- 家人可能生气,但会帮你
- 拖下去只会更糟
- 7个自杀的受害者，都是因为没告诉家人

**如何开口？**
"爸妈，我遇到了困难
我被骗了，借了一些钱还不上
我很害怕，但我不想瞒你们了
我需要你们的帮助..."

**第3步：保护证据**
```
- 截图所有聊天记录
- 导出转账记录
- 录音催收电话
- 保存合同照片
- 记录催收时间和方式

重要：
- 不要删除任何信息
- 云端备份（防止对方远程删除）
```

**第4步：法律援助**
```
渠道1：学校法律援助中心
- 大部分高校都有
- 免费咨询
- 专业律师

渠道2：当地法律援助中心
- 拨打12348
- 学生可申请免费援助

渠道3：中国法律服务网
- 网址：www.12348.gov.cn
- 在线咨询
```

#### 🟡 一般情况（欠款但未被威胁）

**第1步：停止还款（超额部分）**
```
法律规定：
年利率超过36%的部分，不受法律保护

计算公式：
合法利息 = 本金 × 36% ÷ 12 × 月数

举例：
借款10,000元，3个月
合法利息 = 10,000 × 36% ÷ 12 × 3 = 900元

如果对方要你还15,000元：
应还：10,000 + 900 = 10,900元
多出的：15,000 - 10,900 = 4,100元（不用还）
```

**第2步：整理债务清单**
```
列表：
平台A：借10,000元，已还3,000元，还欠?元
平台B：借8,000元，已还2,000元，还欠?元
...

重点：
- 分清本金和利息
- 计算合法利息
- 识别虚增部分
```

**第3步：协商还款**
```
话术：
"你好，我是学生，确实还不起了"
"根据法律，超过年利率36%的部分我不用还"
"我愿意还本金+合法利息"
"我可以分期，但你们要停止催收"
"如果你们不同意，我就报警"

重点：
- 录音
- 书面确认（微信、短信）
- 不承认虚增债务
```

#### 🟢 预防阶段（还没借但有想法）

**自查清单：**
```
❓ 我真的需要这笔钱吗？
❓ 我有能力还吗？（不依赖"以后"）
❓ 这个平台正规吗？（银保监会可查）
❓ 我告诉父母了吗？（如果不敢说，就别借）
❓ 我看合同了吗？（每一页都看）
❓ 利息是多少？（超过24%就别碰）
❓ 有砍头息吗？（有就是诈骗）
❓ 有备用方案吗？（如果还不上怎么办）
```

**如果8个问题有3个以上是"否"或"不确定"**
**→ 不要借！**
