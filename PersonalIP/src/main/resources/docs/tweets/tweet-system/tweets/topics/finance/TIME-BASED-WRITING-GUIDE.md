# 金融推文时间驱动创作指南 ⏰

> **核心原则**: 金融市场瞬息万变，内容必须跟随时间节奏

---

## 🚨 为什么时间如此重要？

### 金融内容的时效性特点：

1. **价值衰减极快**
   - 数据公布前：高价值（前瞻分析）
   - 数据公布后1小时：极高价值（第一时间解读）
   - 数据公布后24小时：中等价值（深度分析）
   - 数据公布后1周：低价值（已成旧闻）

2. **市场情绪变化快**
   - 周一开盘：焦虑/期待
   - 周三数据日：紧张/兴奋
   - 周五收盘：放松/总结
   - 周末：反思/规划

3. **竞争激烈**
   - 同一事件，晚发布2小时 = 失去先机
   - 马后炮分析 = 失去credibility

## 📅 时间驱动创作的3个步骤

### Step 1: 查看日历（每天必做）

**早上起床第一件事**:
```bash
打开 CONTENT-CALENDAR.md
```

检查：
- [ ] 今天是几号？星期几？
- [ ] 今天有没有重要数据发布？
- [ ] 本周的市场焦点是什么？
- [ ] 有没有突发事件需要响应？

### Step 2: 市场扫描（5分钟）

**快速浏览**:
- Bloomberg首页
- CNBC市场新闻
- TradingView价格变化
- Twitter Finance热门话题

**记录要点**:
```
今日市场：
- 标普: _____ (±__%)
- 纳指: _____ (±__%)
- 黄金: $_____ (±__%)
- BTC: $_____ (±__%)

热点新闻:
1. _______________
2. _______________
3. _______________
```

### Step 3: 选择主题

**优先级排序**:

🔴 **最高优先级：突发事件（立即响应）**
- 市场暴跌/暴涨 (±3%)
- 重大政策发布
- 黑天鹅事件
- **响应时间**: 2小时内

🟡 **高优先级：今日数据（当天发布）**
- CPI、非农、GDP等重要数据
- 美联储决议
- 财报季关键公司
- **响应时间**: 数据公布后1-4小时

🟢 **正常优先级：日历计划（按计划）**
- 周一展望
- 周五复盘
- 周末深度分析
- **响应时间**: 当天内

⚪ **低优先级：常青内容（随时）**
- 投资方法论
- 交易心理
- 教育内容
- **响应时间**: 不限

## ⚠️ 常见错误

### ❌ 错误1：完全忽略时间

**案例**:
周五发布"本周市场展望"
→ 周都快过完了，还展望什么？

**正确做法**:
周一早上发布展望

---

### ❌ 错误2：马后炮分析

**案例**:
数据公布3天后："我早就说了会这样"
→ 那你当时怎么不说？

**正确做法**:
有观点就在事前/事中表达

---

### ❌ 错误3：错过最佳时机

**案例**:
美联储决议后24小时才发分析
→ 别人都分析透了

**正确做法**:
决议后2-4小时内发布

---

### ❌ 错误4：内容与时间不符

**案例**:
周五晚上发复杂的技术分析
→ 大家准备周末了，谁看这个？

**正确做法**:
周五晚发轻松内容/总结

## 💡 时间管理技巧

### 技巧1：批量准备

**周末花2小时**:
- 准备下周5个推文框架
- 只差填充最新数据
- 大大提高平时效率

### 技巧2：模板化

**为不同类型内容建立模板**:
- 数据解读模板 ✓
- 行情分析模板 ✓
- 周度复盘模板 ✓
- 突发事件模板 ✓

### 技巧3：设置提醒

**日历提醒**:
- 重要数据前一天
- 美联储会议日期
- 财报季时间
- 每周日更新提醒

### 技巧4：建立快速响应机制

**突发事件SOP**:
```
接到消息（5分钟内）
    ↓
确认信息来源
    ↓
快速发布简短观点（15分钟内）
    ↓
准备深度分析（1-2小时内）
    ↓
发布完整分析
    ↓
持续跟进市场反应
```
