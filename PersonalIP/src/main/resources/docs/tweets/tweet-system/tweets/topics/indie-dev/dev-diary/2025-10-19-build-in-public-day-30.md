# Build in Public Day 30: 第一个付费用户！

## 推文内容

今天收到第一笔付款通知 💰
$19/月

金额不大，意义重大

**数据回顾（30天）：**
- 用户：285
- 付费转化：1（0.35%）
- MRR：$19
- 日活：42（15%）

**学到的：**

1. **免费用户≠付费用户**
   285个注册，只有1个付费
   转化率比预期低

2. **要主动沟通**
   这个付费用户
   我发邮件问了需求
   针对性优化功能
   然后他付费了

3. **定价很关键**
   $19可能太低
   下月测试$29

**下个月目标：**
- 3个付费用户
- $100 MRR
- 优化onboarding

继续分享过程 🚀
