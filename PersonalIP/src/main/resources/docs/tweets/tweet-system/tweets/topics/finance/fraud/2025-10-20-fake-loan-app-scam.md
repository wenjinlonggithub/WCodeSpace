# 714高炮：借3000还10万的噩梦

## 推文内容

一个大学生借款3000元应急，3个月后欠债10万，最后跳楼自杀。

这不是电影，这是真实发生的"714高炮"案例。

**案例：套路贷APP诈骗**

**受害者：小张（化名），21岁大学生**

**诈骗流程全记录：**

**第1天：下载APP借款**
- 急需3000元交学费
- 搜索"快速借款"下载APP
- 填写身份信息、通讯录、位置权限
- 到账2100元（砍头息：扣除900元"服务费"）

实际借款：3000元
实际到账：2100元

**第7天：还款日到来**
- 需还款3000元 + 利息1050元 = 4050元
- 周利息35%，年化利率1820%（！）
- 还不上，客服"贴心"推荐其他平台

**第30天：以贷养贷循环**
- 已在8个平台借款
- 总借款：3万元
- 实际到账：1.8万元
- 应还金额：5.2万元

**第90天：崩盘**
- 应还总额：10.8万元
- 爆通讯录：父母、同学、老师都接到催收电话
- P图威胁：合成不雅照片威胁传播
- 暴力催收：24小时骚扰电话

小张最终选择跳楼...

**"714高炮"的5个特征：**

🚩 7天或14天为借款周期
🚩 砍头息：实际到账远低于借款金额
🚩 超高利息：年化利率超过500%
🚩 要求授权通讯录、照片等隐私权限
🚩 诱导"以贷养贷"形成死循环

**自救方法：**
✅ 报警：年利率超过36%即为违法
✅ 保留证据：聊天记录、转账记录
✅ 投诉：银保监会、互联网金融举报平台
✅ 法律援助：只需归还本金+合法利息

记住：**合法民间借贷年利率上限15.4%（2024年标准）**

超过这个数字的，都不用还！
