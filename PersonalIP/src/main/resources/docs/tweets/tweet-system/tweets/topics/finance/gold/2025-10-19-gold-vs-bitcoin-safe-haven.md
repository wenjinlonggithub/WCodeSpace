# 黄金 vs 比特币：谁才是真正的避险资产？

## 推文内容

当美联储放水时，黄金和比特币都在涨
当加息周期来临，它们又一起跌

有人说比特币是"数字黄金"
但数据告诉我：它们根本不是一回事 🤔

**黄金**：
- 5000年历史验证
- 波动率：年化15-20%
- 央行储备资产
- 真正的避险属性

**比特币**：
- 15年历史
- 波动率：年化60-80%
- 风险资产属性
- 更像科技股

真正的区别在哪？

当地缘政治危机爆发时：
黄金 ↗️（避险）
比特币 ↘️（风险资产遭抛售）

当流动性宽松时：
黄金 ↗️（通胀对冲）
比特币 ↗️↗️↗️（流动性最敏感资产）

我的配置逻辑：
- 黄金 = 保险，应对黑天鹅
- 比特币 = 进攻，把握周期机会

它们不是替代关系，而是互补
