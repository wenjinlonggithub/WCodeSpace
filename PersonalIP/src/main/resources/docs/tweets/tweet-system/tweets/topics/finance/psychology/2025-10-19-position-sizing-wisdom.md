# 仓位管理：比选股更重要的技能

## 推文内容

买对了股票，却没赚到钱
为什么？

因为仓位太小

买错了股票,账户爆仓
为什么？

因为仓位太大

**仓位管理，决定交易的生死** ⚖️

做了5年交易，我总结出一套仓位管理体系
分享给你们：

**1. 金字塔仓位模型**

我的资金分配：
- 40% 核心仓位（长期持有，低频交易）
- 30% 机会仓位（趋势交易，中等频率）
- 20% 灵活仓位（短线交易，高频率）
- 10% 现金储备（应对突发机会）

不同仓位，不同目标

**2. 单笔交易仓位上限**

我的铁律：
- 单笔交易≤ 总资金的5%
- 单个品种 ≤ 总资金的15%
- 相关品种合计 ≤ 总资金的25%

为什么？

假设你有100万
单笔亏损5% = -5万
连续3次错误 = -15万
还剩85万，有能力翻身

但如果单笔30%
一次错误 = -30万
两次错误 = -60万
心态崩了，账户也崩了

**3. 确定性越高，仓位越大**

这是关键！

我的仓位等级：
- 试探性仓位：2%（不确定，先试试）
- 标准仓位：5%（符合系统信号）
- 加重仓位：8%（高胜率机会）
- 重仓：10-15%（千载难逢，极少用）

举例：

2024年3月比特币ETF通过后回调
我判断：长期向上，短期超跌
这是高确定性机会

分批建仓：
$58k → 3%仓位
$52k → 5%仓位
$48k → 7%仓位

总计15%，成本$52k
现在$64k，盈利23%

但如果一开始全部15%买在$58k？
遇到回调至$48k，浮亏17%
大多数人已经被洗出去了

**4. 浮盈加仓，浮亏绝不加仓**

错误示范：
买入后跌了，觉得"更便宜了"
继续加仓摊低成本
结果越陷越深

正确做法：
买入后涨了，趋势确认
浮盈基础上加仓
亏了就认错

记住：
市场不欠你反弹
下跌往往还会继续跌

**5. 盈亏比决定仓位大小**

我的计算公式：
仓位% = 2% × 盈亏比

举例：
止损10%，目标30%
盈亏比 = 3:1
仓位 = 2% × 3 = 6%

止损5%，目标15%
盈亏比 = 3:1
仓位 = 2% × 3 = 6%

如果盈亏比 < 2:1
我根本不交易

**6. 极端行情降低仓位**

VIX恐慌指数 > 30（市场恐慌）
总仓位降至50%以下

市场过于亢奋（所有人都在谈股票）
总仓位降至60%以下

为什么？
极端行情难预测
减仓 = 降低不确定性

**我的惨痛教训：**

2022年底，我判断BTC会涨
但用了40%仓位重仓
结果遇到FTX暴雷，跌30%

账户浮亏12%，心态崩溃
最后$18k被迫止损
两周后，BTC开始反弹...

如果当时只用10%仓位
浮亏3%，完全可以承受
也不会被洗出去

**这件事教会我：**

仓位管理 > 判断方向

对的方向 + 错的仓位 = 亏钱
错的方向 + 对的仓位 = 小亏
对的方向 + 对的仓位 = 大赚

**给新手的3条建议：**

1. 开始用2-3%仓位练手
等能稳定盈利，再提高到5%

2. 永远不要满仓
留现金，才有机会

3. 记录每笔交易的仓位
复盘时分析：仓位是否合理？

**记住这句话：**

仓位管理的本质
不是让你赚更多
而是让你活更久

活得久，才能等到大机会 🎯

你的仓位管理系统是什么？
