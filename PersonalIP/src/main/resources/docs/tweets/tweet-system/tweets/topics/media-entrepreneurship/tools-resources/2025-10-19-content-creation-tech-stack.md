# 我的内容创作工具栈：从拍摄到发布的完整workflow

## 推文内容

每天有人问我：
"你用什么工具创作内容？"

今天完整公开我的tech stack 🛠️

**设备（硬件）**

📱 拍摄设备：
- 主力：iPhone 14 Pro（$999）
- 备用：Sony ZV-1（$698）
- 三脚架：Joby GorillaPod（$35）

真相：
80%的视频用iPhone拍
相机只在需要专业感时用

建议新手：
手机完全够用，别被器材困住

💻 剪辑设备：
- MacBook Pro M2（$2499）
- iPad Air（脚本用，$599）

**软件（核心工具）**

🎬 视频剪辑：
- CapCut（免费）← 主力
- Final Cut Pro（$299）← 复杂项目
- Descript（$24/月）← 自动字幕

为什么选CapCut？
- 免费且功能强大
- 自动识别字幕
- 模板丰富
- 导出无水印

**我的剪辑workflow：**
1. 导入素材
2. 自动生成字幕（准确率90%）
3. 用模板调色
4. 加转场和特效
5. 导出

15分钟搞定一条短视频

📝 脚本创作：
- Notion（$8/月）← 内容管理
- ChatGPT Plus（$20/月）← 灵感生成
- Grammarly（$12/月）← 语法检查

Notion模板：
```
标题创意 → 脚本大纲 → 逐字稿 → 拍摄提示 → 发布记录
```

所有内容都在一个database

🎨 设计工具：
- Canva Pro（$12.99/月）← 封面图
- Figma（免费）← 复杂设计
- Remove.bg（免费）← 抠图

Canva使用率90%
简单、快速、够用

📊 数据分析：
- Google Analytics（免费）
- TikTok Analytics（免费）
- Notion（整合所有数据）

我的数据追踪表：
- 发布时间
- 播放量
- 互动率
- 涨粉数
- 主题标签

每周分析，找规律

🔍 关键词研究：
- Google Trends（免费）
- TikTok搜索框（免费）
- AnswerThePublic（免费）

找热点话题的3个方法：
1. Google Trends看上升趋势
2. TikTok搜索框看联想词
3. AnswerThePublic看用户问题

⏰ 定时发布：
- Buffer（$6/月）← X/LinkedIn
- Later（$25/月）← Instagram/TikTok
- Native定时（免费）← YouTube

为什么用工具？
批量准备内容
固定时间发布
提高发布一致性

🎵 音乐素材：
- Epidemic Sound（$13/月）
- Artlist（$14.99/月）
- TikTok Sound Library（免费）

版权很重要！
用免费音乐可能被muted

**我的完整workflow：**

**周日（批量策划）**
- Notion里列出下周7个选题
- ChatGPT辅助生成大纲
- 准备拍摄清单

**周一-周三（集中拍摄）**
- 批量拍摄3-5条素材
- iPhone + 三脚架
- 每条拍3遍选最好的

**周四-周五（剪辑）**
- CapCut批量剪辑
- Canva做封面
- Notion记录发布计划

**周六（发布+互动）**
- Buffer定时发布
- 前3小时密集回复评论
- 数据记录到Notion

**每天（碎片时间）**
- 收集内容灵感
- 回复粉丝私信
- 查看数据变化

**成本分析：**

必须付费：
- Notion：$8/月
- CapCut：免费
- Canva：$12.99/月
- 音乐素材：$13/月
**总计：$33.99/月**

可选付费：
- ChatGPT Plus：$20/月
- 定时发布工具：$6-25/月
- 其他工具：$0-50/月

**我的月工具成本：$80**
收入：$2000+
ROI：25倍

**给新手的建议：**

**第1个月（免费工具）**
- CapCut（剪辑）
- Canva免费版（设计）
- Notion免费版（管理）
总成本：$0

先验证能否坚持

**第3个月（基础付费）**
- Canva Pro
- Notion
- 音乐素材库
总成本：$34/月

开始专业化

**第6个月（全套工具）**
根据需求补齐
总成本：$50-100/月

**避开的坑：**

❌ 一开始买一堆工具
大部分用不上

❌ 追求最贵的设备
内容 > 器材

❌ 订阅太多软件
选1-2个深度使用

**我的工具选择原则：**

1. 能免费就免费
2. 必须付费选性价比最高
3. 工具服务内容，不是炫耀
4. 简单够用 > 功能强大

**记住：**

工具是放大器
你的创意才是核心

$0工具 + 好内容 > $1000工具 + 烂内容
