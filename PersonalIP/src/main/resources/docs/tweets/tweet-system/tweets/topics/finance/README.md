# 金融投资专题 💰

专注于金融市场分析与投资心得分享。

## 📊 内容方向

### 1. 市场行情分析
- **黄金 (Gold)**: 技术分析、基本面分析、趋势判断
- **标普500 (S&P 500)**: 美股大盘走势、板块轮动、市场情绪
- **纳斯达克 (NASDAQ)**: 科技股分析、创新企业追踪
- **比特币 (Bitcoin)**: 加密货币市场、链上数据分析、宏观环境影响

### 2. 事件分析
- 重大财经事件解读
- 央行政策影响分析
- 地缘政治对市场的影响
- 经济数据解读（CPI、非农、利率决议等）

### 3. 交易心理分析
- 投资心态管理
- 情绪控制技巧
- 认知偏差避免
- 风险管理策略
- 仓位管理心得

### 4. 投资策略分享
- 长期投资理念
- 短期交易技巧
- 资产配置建议
- 技术指标运用

## 📝 推文类型

### 市场观点类
```markdown
格式：观点 + 数据支撑 + 风险提示
示例：对某个品种的行情判断、技术位分析
```

### 事件解读类
```markdown
格式：事件概述 + 影响分析 + 应对策略
示例：美联储议息会议解读、重大经济数据分析
```

### 心理感悟类
```markdown
格式：经历/场景 + 反思 + 启发
示例：交易失败后的反思、心态调整经验
```

### 知识科普类
```markdown
格式：概念解释 + 实例说明 + 应用建议

```

## ⚠️ 内容原则

1. **客观中立**: 分享观点而非荐股，避免绝对化表述
2. **风险提示**: 任何投资建议都需强调风险
3. **数据支撑**: 观点需有数据或逻辑依据
4. **合规合法**: 遵守金融内容发布规范
5. **教育导向**: 帮助读者提升认知，而非盲目跟单

## ⚠️ 创作金融类推文必读

**每次写金融类推文前，必须先完成以下步骤：**

1. **📅 查看 `CONTENT-CALENDAR.md`**
   - 确认今天的日期和重要事件
   - 检查本周/本月的市场焦点
   - 根据时间表选择合适主题

2. **📋 遵循 `WRITING-WORKFLOW.md`**
   - 完成创作前检查清单
   - 使用对应的内容模板
   - 进行发布前质量检查

3. **🔄 每周日更新时间表**
   - 更新下周重要数据/事件
   - 规划下周内容主题
   - 准备相关素材

**记住：金融内容的价值 = 时效性 × 准确性 × 洞察力**
