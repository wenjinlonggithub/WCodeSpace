# 决策框架：如何做出更好的选择

## 🇨🇳 中文版

优秀的人生是一系列优秀决策的复利。

大多数人的误区：
❌ 依靠直觉做决策就够了
❌ 收集更多信息总是更好
❌ 大小决策都要深思熟虑

真相是：
✅ 决策质量>决策数量
✅ 框架化决策减少认知负担
✅ 不同决策需要不同方法

**为什么重要？**

原因1：贝索斯：可逆决策快速做，不可逆决策慢慢做
原因2：决策疲劳真实存在：每天决策能力有限
原因3：好框架节省时间，提升质量

**我的亲身经历：**

之前：大小决策都纠结，花大量时间在无关紧要的选择上
现在：用框架快速过滤，把精力集中在10%的关键决策
改变：建立决策系统后，决策速度提升3倍，后悔率下降70%

**具体方法拆解：**

技巧1：决策矩阵分类
- Type 1（不可逆）：仔细研究，咨询专家，慢慢做
- Type 2（可逆）：快速决策，错了可以改
- 减少Type 2决策时间，增加Type 1决策质量

技巧2：10-10-10法则
- 这个决策10分钟后的影响？
- 10个月后的影响？
- 10年后的影响？
- 时间维度帮你看清真正重要的事

技巧3：决策日志
- 记录重要决策：决策内容+理由+预期
- 3个月后回顾：实际结果如何？
- 分析偏差：哪些判断是对的？哪些错了？为什么？
- 不断校准决策算法

**实施建议：**
今天开始区分决策类型：小决策2分钟搞定，大决策值得花一周。

**核心方法：**
📝 艾森豪威尔矩阵：重要紧急四象限分类
📝 概率思维：用"60%可能性"替代"一定"和"绝不"
📝 逆向思维：先想最坏情况，能接受吗？

**实践步骤：**
**Day 1-7：** 观察一周：我在哪些决策上花了过多时间？
**Day 8-30：** 应用框架：用10-10-10和Type1/2分类决策
**Day 31+：** 建立决策日志，追踪和改进决策质量

**常见障碍及应对：**
🚧 害怕做错决策 → 💡 记住：不决策也是一种决策，往往是最差的
🚧 信息永远不够 → 💡 设定deadline：X时间内收集，然后必须决策
🚧 被沉没成本困住 → 💡 专注未来收益，过去投入已无法改变

**推荐书籍/资源：**
📚 《清单革命》阿图·葛文德 - 系统化决策
📚 《思考，快与慢》- 决策中的认知偏差
📚 《原则》瑞·达利欧 - 决策算法

你在实践哪个方法？💬
