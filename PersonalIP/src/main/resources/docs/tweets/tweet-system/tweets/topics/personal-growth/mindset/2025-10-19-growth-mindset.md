# 从Fixed Mindset到Growth Mindset的转变

## 推文内容

2年前的我：
"我不是那块料"
"天赋比努力重要"
"失败=能力不行"

现在的我：
"暂时还不会"
"能力可以培养"
"失败=学习机会"

**什么改变了我？**

读了《Mindset》后的5个转变：

**1. 对待失败**

Fixed: 失败=我不行
Growth: 失败=反馈

失败不是终点
是学习的开始

**2. 对待努力**

Fixed: 需要努力=没天赋
Growth: 努力=掌握之路

天才也要努力
努力让天赋发光

**3. 对待批评**

Fixed: 批评=人身攻击
Growth: 批评=成长机会

不是所有批评都对
但都值得思考

**4. 对待他人成功**

Fixed: 他人成功威胁我
Growth: 他人成功激励我

别人强不代表你弱
学习+竞争=成长

**5. 对待挑战**

Fixed: 避免挑战保面子
Growth: 拥抱挑战求成长

Comfort zone外才有成长

**实践方法：**

1. 语言转变
   "我不会"→"我还不会"
   "太难了"→"需要时间"
   
2. 记录成长
   每周写：
   - 学到什么
   - 克服什么困难
   - 下周挑战

3. 拥抱不完美
   发布60分作品
   边做边改进

4. Seek feedback
   主动要批评
   不是夸奖

**最大改变：**

以前：怕被看不起
现在：勇于尝试

以前：perfectionist
现在：iterationist

以前：证明自己
现在：提升自己

**记住：**

能力不是固定的
是可以成长的

问题不是"我能不能"
而是"我如何能"

你是哪种mindset？
