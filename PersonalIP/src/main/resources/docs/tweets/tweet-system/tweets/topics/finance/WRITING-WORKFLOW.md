## 📋 创作前准备（必做）

### Step 1: 时间确认
```
当前日期: ___________
当前时间: ___________
星期几: ___________
市场状态: [ ] 开盘 [ ] 盘中 [ ] 收盘 [ ] 休市
```

### Step 2: 查看内容日历
打开 `CONTENT-CALENDAR.md`

检查项:
- [ ] 今天是否有重要数据发布？
- [ ] 本周关键事件是什么？
- [ ] 今天推荐的内容主题是什么？
- [ ] 有没有突发事件需要响应？

### Step 3: 市场扫描（5分钟）

**必看数据**:
- [ ] 标普500当日/周涨跌: ______%
- [ ] 纳斯达克当日/周涨跌: ______%
- [ ] 黄金价格: $______
- [ ] 比特币价格: $______
- [ ] 美元指数: ______
- [ ] 10年期美债收益率: ______%

**最新新闻**:
- Bloomberg头条: _______________
- CNBC头条: _______________
- 加密货币新闻: _______________

### Step 4: 确定主题

根据以上信息，今天写什么？

选择标准（优先级排序）:
1. 🚨 **突发重大事件** - 立即响应
2. 📊 **今日重要数据** - 当天发布
3. 📅 **日历计划主题** - 按计划执行
4. 💡 **市场热点话题** - 蹭热度
5. 📚 **教育类内容** - 填补空白

## 🔍 内容质量检查清单

**发布前必检**:

### 时效性检查
- [ ] 数据是最新的吗？
- [ ] 有没有更新的消息出来？
- [ ] 内容是否还relevant？

### 准确性检查
- [ ] 数字核对过了吗？
- [ ] 专业术语用对了吗？
- [ ] 有没有事实错误？
- [ ] 引用有来源吗？

### 价值性检查
- [ ] 提供了独特视角吗？
- [ ] 有可执行的建议吗？
- [ ] 对读者有帮助吗？
- [ ] 不是马后炮吧？

### 风险检查
- [ ] 添加风险提示了吗？
- [ ] 有没有绝对化表述？
- [ ] 是分享观点，不是荐股？
- [ ] 符合合规要求吗？

### 互动性检查
- [ ] 开头有hook吗？
- [ ] 结尾有提问吗？
- [ ] 内容引发思考吗？
- [ ] 鼓励评论了吗？

### 格式检查
- [ ] 标题吸引人吗？
- [ ] 段落清晰吗？
- [ ] 数据可视化了吗？
- [ ] 标签合适吗？

## ⚡ 快速响应流程（突发事件）

**重大市场波动时**:

```

    ↓
15分钟：确认事实
    ↓
30分钟：快速分析
    ↓
60分钟：发布初步观点
    ↓
4小时：发布深度分析
    ↓
24小时：跟进市场反应
```
