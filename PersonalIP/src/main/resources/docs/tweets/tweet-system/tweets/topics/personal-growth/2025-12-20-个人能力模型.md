# 个人能力模型：T型人才vs π型人才

## 🇨🇳 中文版

认清自己的能力版图，是一切发展的前提。

大多数人的误区：
❌ 能力就是专业技能
❌ 全面发展最好
❌ 能力是静态的

真相是：
✅ 能力=硬技能+软技能+元能力
✅ T型人才>全能人才
✅ 能力是动态的，需要持续迭代

**为什么重要？**

原因1：哈佛研究：软技能对职业成功贡献85%
原因2：元能力（学习能力、思维能力）是能力的杠杆
原因3：清晰的能力模型指导发展方向

**我的亲身经历：**

之前：盲目学习，什么都懂一点什么都不精
现在：建立T型能力模型：一深多广
改变：聚焦深度专长+拓展关键广度，职业价值倍增

**具体方法拆解：**

技巧1：三层能力金字塔
- 底层：元能力（学习力、思考力、适应力）
- 中层：可迁移技能（沟通、协作、解决问题）
- 顶层：专业技能（行业知识、工具技能）

技巧2：T型能力设计
- 一竖：深度专长（成为前10%）
- 一横：关键广度（理解相关领域）
- 聚焦深度，选择性拓展广度

技巧3：能力盘点法
- 列出所有能力（3个层次）
- 评估当前水平（1-10分）
- 标注价值（对目标的贡献）
- 设计发展路径

**实施建议：**
今天画出你的T型能力图：深度专长是什么？需要哪些广度支撑？

**核心方法：**
📝 元能力优先：底层能力决定上限
📝 T型发展：深度+选择性广度
📝 持续迭代：能力地图每季度更新

**实践步骤：**
**Day 1-7：** 能力盘点：列出你的能力清单
**Day 8-30：** 设计T型：确定深度和广度策略
**Day 31+：** 刻意发展：按计划提升关键能力

**常见障碍及应对：**
🚧 不知道自己有什么能力 → 💡 从过去成就反推能力
🚧 什么都想学 → 💡 聚焦T型，不要追求全能
🚧 能力提升太慢 → 💡 复利效应需要时间，坚持深耕

**推荐书籍/资源：**
📚 《你的降落伞是什么颜色》- 能力自我认知
📚 《优势》唐纳德·克利夫顿 - 发现优势能力
📚 《深度工作》- 建立深度能力

你在实践哪个方法？💬
