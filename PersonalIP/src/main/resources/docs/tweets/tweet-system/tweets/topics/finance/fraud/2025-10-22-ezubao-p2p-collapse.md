# e租宝：762亿P2P骗局的覆灭

## 推文内容

一个打着"互联网金融创新"旗号的平台,如何在18个月内骗走90万投资人762亿元?

**案例:e租宝P2P平台诈骗案(2014-2015)**

**受害者:**
- 投资人数:90万+
- 涉案金额:762亿元人民币
- 实际损失:约600亿(部分被挥霍)
- 平均投资:8.4万元/人

**平台包装:**
- 号称"A股上市公司背景"(虚假)
- 央视、卫视广告轰炸
- 线下体验店遍布全国
- 承诺年化收益9%-14.6%

**骗局设计:**

**第1步:制造真实项目假象**
宣称业务模式:
"我们把融资租赁债权转让给投资者"
"项目都有实物资产抵押"
"风险极低,收益稳定"

真相:95%的项目是虚构的!

**第2步:疯狂营销建立信任**
- 2014年广告投入:10亿元
- 明星代言、综艺冠名
- 总部装修豪华,接待专业
- 前期确实按时返息

**第3步:借新还旧庞氏循环**
```
老用户的收益 = 新用户的本金
只要不断拉新,就能维持运转
```

**真实受害者:张阿姨的故事**

2014年7月:看到央视广告,投入5万元试水
2014年10月:收益正常,追加投入15万元
2015年3月:拉老姐妹们一起投,又投20万
2015年6月:账户显示本息共45万

2015年12月8日:平台被查封
张阿姨40万养老钱打水漂,最终追回不到8万...

**e租宝骗局的5个迷惑性手段:**

🎭 **巨额广告营销造假象**
- 央视投放让人觉得"正规可靠"
- 实际上广告商不审核金融资质

🎭 **线下门店增加信任感**
- 全国开设500+体验店
- 有实体店=安全?(大错特错)

🎭 **合同条款专业复杂**
- 密密麻麻的法律文本
- 普通人根本看不懂

🎭 **前期真实返息钓鱼**
- 小额投资可正常提现
- 吸引大额追加投入

🎭 **虚构项目专业包装**
- 假合同、假企业、假项目
- 配合虚假尽调报告

**案件结果(2017年判决):**
- 主犯丁宁判无期徒刑
- 26名高管获刑
- 追缴资产:约300亿元
- 兑付率:约50%(行业少见的高兑付)

**识别P2P骗局的7个信号:**

🚨 年化收益超过10%(已超出正常范畴)
🚨 广告投放过度激进
🚨 承诺保本保息(违反金融常识)
🚨 项目信息不透明
🚨 资金流向不清晰
🚨 过度强调"央视广告""上市背景"
🚨 线下推广拉人头式营销

**防骗自保指南:**

✅ 检查平台是否有ICP备案
✅ 查询工商信息(天眼查、企查查)
✅ 看是否有银行存管(非平台自建账户)
✅ 项目信息必须可核查
✅ 不被高收益迷惑(超过8%就要警惕)

**一句话总结:**
央视广告≠安全可靠
上市公司≠绝对保障
高收益=高风险(甚至是骗局)

2015年后,P2P行业进入爆雷潮,超过6000家平台关闭,涉案金额数千亿。

如果有人推荐你"稳赚高收益"的理财产品,请想想e租宝的90万受害者。
