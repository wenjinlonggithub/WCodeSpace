# 10个VS Code技巧让我提效50%

## 推文内容

用了5年VS Code才发现这些功能 🤯

**1. Multi-Cursor Selection**
Alt+Click 多光标
Ctrl+D 选中相同单词
批量修改变量名：3秒搞定

**2. Zen Mode**
Ctrl+K Z 进入专注模式
隐藏所有干扰
写代码效率翻倍

**3. Go to Definition**
F12 跳转定义
Alt+← 返回
不用再上下滚动找代码

**4. Command Palette**
Ctrl+Shift+P
所有功能都在这
比记快捷键快

**5. Integrated Terminal**
Ctrl+` 打开终端
不用切换窗口
workflow丝滑

**6. Code Snippets**
自定义代码片段
输入缩写+Tab
完整模板出来

我的snippets:
- `cl` → console.log()
- `rfc` → React Function Component
- `test` → Jest test template

**7. Git Integration**
直接在编辑器：
- 查看diff
- Stage changes
- Commit
- Push

不用开Git GUI

**8. Settings Sync**
同步配置到云端
换电脑无缝切换
扩展+快捷键+主题全同步

**9. Remote Development**
SSH连接服务器
像本地一样编辑
告别vim痛苦

**10. Extensions必装：**
- ESLint（代码检查）
- Prettier（格式化）
- GitLens（Git增强）
- Auto Rename Tag（HTML）
- Better Comments（注释高亮）

**我的快捷键清单：**

导航：
- Ctrl+P 文件跳转
- Ctrl+Shift+O 符号跳转
- Ctrl+G 跳转到行

编辑：
- Ctrl+D 选中下一个
- Ctrl+Shift+L 选中所有
- Alt+↑/↓ 移动行
- Ctrl+/ 注释

重构：
- F2 重命名
- Ctrl+. 快速修复

**效率对比：**

以前：
找函数定义：滚动2分钟
改变量名：手动一个个改
格式化代码：手动调整

现在：
F12 秒跳
Ctrl+D全选+改
Save自动格式化

节省的时间用来：
- 早下班
- 学新技术
- 摸鱼
