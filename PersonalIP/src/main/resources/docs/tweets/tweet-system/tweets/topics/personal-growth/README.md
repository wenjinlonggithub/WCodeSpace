## 📝 推文类型

### 思考感悟类
```markdown
格式：观察/经历 + 思考 + 启发/行动
示例：今天意识到一个关于 XX 的认知盲区...
```


## ⚠️ 内容原则

1. **真诚为先**: 分享真实经历，不贩卖焦虑
2. **可操作性**: 提供具体方法，而非空洞道理
3. **持续性**: 成长是长期过程，保持更新
4. **尊重差异**: 承认个体差异，不强加观点
5. **避免说教**: 分享而非教导，引发思考而非灌输

## 🎓 内容框架参考

### STAR 故事框架
- **Situation**: 当时的情境
- **Task**: 面临的任务/挑战
- **Action**: 采取的行动
- **Result**: 最终的结果和收获

### 黄金圈法则 (Why-How-What)
- **Why**: 为什么这么做/为什么重要
- **How**: 如何做到的/方法是什么
- **What**: 具体是什么/成果如何

### 问题-解决方案框架
- **问题**: 遇到的困境/痛点
- **尝试**: 探索过的方法
- **发现**: 有效的解决方案
- **应用**: 如何实践和推广
