Confidence isn't innate, it's accumulated through small wins

True confidence comes from: I know I can do it, because I've done similar things.

Scientific confidence building:

1. Small Wins Strategy
   Set goals slightly above current ability
   Celebrate completion, reinforce 'I can' belief
   Gradually increase difficulty

2. Evidence Collection
   Record daily small achievements
   Build 'success journal' for review
   Use facts to combat impostor syndrome

3. Body Language
   Posture, eye contact affect mental state
   Power Pose 2min boosts confidence hormones

4. Skill Development
   Deep dive one field to top 10%
   Professional competence is confidence foundation

Remember: Confidence = Well-prepared + Sufficient evidence
