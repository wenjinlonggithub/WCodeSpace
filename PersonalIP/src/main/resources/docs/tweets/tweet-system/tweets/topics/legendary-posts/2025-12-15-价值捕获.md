# 价值捕获：创造价值容易，捕获价值难

## 🇨🇳 中文版

大家都在追求"创造价值"。

但真相是：这个世界上充满了创造巨大价值却一无所获的人——价值捕获能力，才是区分穷人思维和富人思维的分水岭。

为什么？

Peter Thiel在《从0到1》中写道："创造价值不等于捕获价值。完美竞争市场中，所有价值都被消费者捕获了，生产者一分钱都赚不到。"航空业就是最好的例子——为人类创造了万亿价值，但100年来总体亏损。

**震撼的价值捕获对比：**

1. **Google vs 报纸业：**

• **Google的价值捕获率：**
  - 为世界创造的价值：每年帮用户节省的时间价值 ≈ 5万亿美元（假设每次搜索节省10分钟，每分钟价值1美元）
  - Google捕获的价值：2023年广告收入2380亿美元
  - 价值捕获率：2380亿 ÷ 5万亿 = 4.8%
  - 即使只捕获了不到5%的价值，已经让Google成为万亿市值公司

• **报纸业的价值崩溃：**
  - 1990年代，美国报纸业年广告收入：650亿美元
  - 2023年，美国报纸业年广告收入：80亿美元（下降88%）
  - 报纸创造的价值（新闻、信息）并未减少，但价值被Google、Facebook捕获了
  - 这就是"创造价值但无法捕获"的悲剧

2. **苹果vs微软（不同时代的价值捕获策略）：**

• **1990年代的微软：捕获操作系统的价值**
  - Windows授权费：每台PC收取50-100美元
  - 全球PC出货量：每年3亿台
  - 微软年收入：150-300亿美元仅来自Windows授权
  - 价值捕获率极高：PC产业链的大部分利润被微软拿走

• **2010年代的苹果：捕获硬件+生态系统的价值**
  - iPhone硬件利润：平均每台150-200美元
  - App Store抽佣30%：2023年收入850亿美元
  - 苹果捕获了智能手机产业链66%的利润（全球智能手机品牌超过100个，苹果市场份额18%，利润份额66%）
  - 这是价值捕获效率的极致体现

3. **特斯拉vs传统车厂：不同的价值捕获机制**

• **传统车厂：一次性销售模式**
  - 丰田：卖一辆车，赚5000-8000元，结束
  - 通用：卖一辆车，赚3000-5000元，结束
  - 客户终身价值 = 购车价格 × 15% （毛利率）

• **特斯拉：持续价值捕获**
  - 卖车：每辆赚8000-12000美元
  - FSD订阅：每月199美元 × 10年 = 23880美元
  - 超级充电网络：每年1000-2000美元
  - 保险服务：每年1200-1500美元
  - OTA软件升级：未来可能按功能收费
  - 客户终身价值 = 50000-80000美元（是传统车厂的5-8倍）

**价值捕获的五大机制：**

1. **垄断性捕获（定价权）：**

• **OPEC的石油定价权：**
  - 控制全球40%的石油产量
  - 通过减产/增产操控油价
  - 1970年代油价从3美元/桶飙升到30美元/桶，OPEC捕获了石油价值的90%
  - 对比：页岩油公司虽然创造价值（增加供应），但无定价权，油价低迷时大量破产

• **钻石行业的戴比尔斯：**
  - 控制全球钻石供应的70%（1990年代）
  - 通过"钻石恒久远，一颗永流传"营销，创造了"钻石=爱情"的心理锁定
  - 钻石的实际稀缺性远低于宣传，但价格被人为维持在高位
  - 价值捕获机制：控制供应+控制叙事

2. **平台抽佣（流量税）：**

• **App Store的30%税：**
  - 开发者创造价值（开发APP）
  - 苹果捕获价值（抽佣30%）
  - 2023年App Store总收入：8500亿美元GMV，苹果捕获850亿美元
  - Epic Games起诉苹果垄断，但败诉——因为苹果控制了"分发渠道"

• **外卖平台的佣金：**
  - 餐厅创造价值（做饭）
  - 美团/饿了么捕获价值（抽佣20-25%）
  - 餐厅净利润率本来就只有10-15%，外卖平台抽走20%，很多餐厅亏本做外卖
  - 但餐厅无法不做外卖——因为流量在平台手里

3. **数据锁定（切换成本）：**

• **Salesforce的数据囚笼：**
  - 企业在Salesforce里存了10年客户数据
  - 切换到其他CRM？数据迁移成本 = 500万-5000万美元
  - 即使竞品免费，企业也不敢切换
  - 价值捕获机制：数据绑架

• **微信的社交数据：**
  - 13亿用户，10年聊天记录、朋友圈、支付记录
  - 切换到其他社交软件？你的社交图谱无法迁移
  - 这就是为什么字节、阿里多次尝试做社交都失败——数据护城河太深

4. **品牌溢价（认知垄断）：**

• **爱马仕的稀缺性定价：**
  - Birkin包制造成本：约1500美元
  - 官方售价：8000-15000美元
  - 二手市场价：15000-30000美元（部分款式）
  - 价值捕获率：85-90%（成本只占售价的10-15%）
  - 机制：人为制造稀缺（限量生产+配货制度）+ 品牌叙事（手工艺+传承）

• **茅台的文化绑定：**
  - 生产成本：约200元/瓶
  - 出厂价：1499元/瓶
  - 市场价：2500-3000元/瓶
  - 价值捕获率：92%（毛利率）
  - 机制：绑定中国商务文化（"谈生意必须喝茅台"）+ 稀缺性（限产）

5. **时间套利（先发优势）：**

• **可口可乐的百年心智占据：**
  - 130年品牌积累
  - "可乐"这个品类 = 可口可乐（品类即品牌）
  - 即使百事可乐味道更好（盲测胜出），市场份额仍远低于可口可乐
  - 价值捕获机制：心智占位（占据"正宗可乐"的认知）

• **腾讯的时间复利：**
  - QQ（1999年）→ 微信（2011年）→ 微信支付（2013年）→ 小程序（2017年）
  - 每一步都建立在前一步的用户基础上
  - 后来者即使产品更好，也无法复制20年的时间积累

**价值捕获的反例（创造价值但无法捕获）：**

• **开源软件：**
  - Linux为世界创造了数万亿价值（服务器、安卓系统基础）
  - Linux基金会年收入：1.7亿美元
  - 价值捕获率 < 0.01%
  - 但Red Hat（提供Linux企业服务）被IBM以340亿美元收购——价值捕获的关键是"服务"而非"软件"

• **航空业：**
  - 1903-2023年，航空业为人类创造了不可估量的价值（缩短了全球距离）
  - 但航空业整体累计亏损（所有航空公司历史利润总和 < 0）
  - 价值被谁捕获了？旅客（低票价）、波音/空客（卖飞机）、机场（起降费）、石油公司（燃油）
  - 航空公司：创造价值，但在产业链中议价能力最弱

• **科学家：**
  - 爱因斯坦的相对论为世界创造了无限价值（GPS、核能、量子物理）
  - 爱因斯坦的财富：中产阶级水平
  - 对比：iPhone利用GPS技术，苹果市值2.8万亿美元
  - 这就是"创造价值"与"捕获价值"的残酷差距

**如何提高价值捕获能力：**

1. **建立定价权：**
   - 通过品牌、专利、网络效应、垄断地位
   - 案例：Nike不生产鞋（代工），但通过品牌捕获了70%的利润

2. **控制关键节点：**
   - 在产业链中占据"不可替代"的位置
   - 案例：Visa/Mastercard不发卡、不放贷，但控制了支付网络，捕获51%净利润率

3. **设计锁定机制：**
   - 数据锁定、生态系统锁定、习惯锁定
   - 案例：苹果通过生态系统（iPhone + Mac + iPad + Watch + AirPods），让客户终身锁定

4. **持续价值捕获模式：**
   - 从"一次性交易"转向"订阅/会员/消耗品"
   - 案例：Adobe从卖软件（一次性收入699美元）转向订阅（每年636美元 × 终身）

5. **认知垄断：**
   - 在客户心智中占据"品类第一"的位置
   - 案例：提到搜索就是Google，提到电商就是亚马逊，提到社交就是Facebook

**可行动建议：**

1. **如果你是创业者：**
   - 在设计产品时，同步设计"价值捕获机制"
   - 不要成为"创造价值但被平台抽佣"的一方，要成为"抽佣的平台"

2. **如果你是投资者：**
   - 不要只看"创造了多少价值"，更要看"捕获了多少价值"
   - 优先投资高价值捕获率的公司：SaaS、平台、品牌、专利

3. **如果你是职场人：**
   - 不要只是"创造价值"（埋头苦干），更要学会"捕获价值"（升职加薪）
   - 建立个人品牌、控制关键资源、成为不可替代

Warren Buffett的终极智慧："宁愿要一个普通的企业和优秀的价值捕获机制，也不要一个伟大的企业和糟糕的价值捕获机制。"

这个认知，颠覆了你对"创造价值"的理解吗？

## 🇬🇧 English Version

# Value Capture: Creating Value Is Easy, Capturing It Is Hard

Everyone chases "value creation."

But the truth is: The world is full of people who create enormous value but gain nothing—value capture ability is the dividing line between poor and rich thinking.

Why?

Peter Thiel wrote in "Zero to One": "Creating value doesn't equal capturing value. In perfect competition, all value is captured by consumers, producers earn nothing." The airline industry is the perfect example—created trillions in value for humanity, but has been net unprofitable over 100 years.

**Shocking Value Capture Comparisons:**

1. **Google vs Newspaper Industry:**

• **Google's Value Capture Rate:**
  - Value created for world: Time saved annually ≈ $5T (assuming each search saves 10 min at $1/min)
  - Value Google captures: 2023 ad revenue $238B
  - Value capture rate: $238B ÷ $5T = 4.8%
  - Even capturing <5% of value created makes Google a trillion-dollar company

• **Newspaper Industry's Value Collapse:**
  - 1990s, US newspaper ad revenue: $65B annually
  - 2023, US newspaper ad revenue: $8B annually (88% decline)
  - Value created by newspapers (news, information) hasn't decreased, but value captured by Google, Facebook
  - This is the tragedy of "creating value without capturing it"

2. **Apple vs Microsoft (Different Era Value Capture Strategies):**

• **1990s Microsoft: Capturing OS Value**
  - Windows licensing fee: $50-100 per PC
  - Global PC shipments: 300M annually
  - Microsoft annual revenue: $15-30B from Windows alone
  - Extremely high value capture rate: Most PC industry profits taken by Microsoft

• **2010s Apple: Capturing Hardware + Ecosystem Value**
  - iPhone hardware profit: Average $150-200 per unit
  - App Store 30% commission: 2023 revenue $85B
  - Apple captures 66% of smartphone industry profits (100+ global smartphone brands, Apple 18% market share, 66% profit share)
  - Ultimate expression of value capture efficiency

3. **Tesla vs Traditional Automakers: Different Value Capture Mechanisms**

• **Traditional Automakers: One-time Sale Model**
  - Toyota: Sell one car, earn $5,000-8,000, done
  - GM: Sell one car, earn $3,000-5,000, done
  - Customer lifetime value = Purchase price × 15% (gross margin)

• **Tesla: Continuous Value Capture**
  - Car sale: Earn $8,000-12,000 per unit
  - FSD subscription: $199/month × 10 years = $23,880
  - Supercharger network: $1,000-2,000 annually
  - Insurance services: $1,200-1,500 annually
  - OTA software updates: Future feature-based pricing possible
  - Customer lifetime value = $50,000-80,000 (5-8x traditional automakers)

**Five Value Capture Mechanisms:**

1. **Monopolistic Capture (Pricing Power):**

• **OPEC's Oil Pricing Power:**
  - Controls 40% of global oil production
  - Manipulates prices through production cuts/increases
  - 1970s oil prices surged from $3/barrel to $30/barrel, OPEC captured 90% of oil value
  - Compare: Shale oil companies create value (increase supply) but have no pricing power, many bankrupt when prices drop

• **Diamond Industry's De Beers:**
  - Controlled 70% of global diamond supply (1990s)
  - Through "A Diamond Is Forever" marketing, created "diamond = love" psychological lock-in
  - Actual diamond scarcity far lower than marketed, but prices artificially maintained high
  - Value capture mechanism: Control supply + control narrative

2. **Platform Commission (Traffic Tax):**

• **App Store's 30% Tax:**
  - Developers create value (build apps)
  - Apple captures value (30% commission)
  - 2023 App Store total revenue: $850B GMV, Apple captures $85B
  - Epic Games sued Apple for monopoly but lost—because Apple controls "distribution channel"

• **Food Delivery Platform Commission:**
  - Restaurants create value (cook food)
  - Meituan/Ele.me capture value (20-25% commission)
  - Restaurants' net margins already only 10-15%, platforms take 20%, many restaurants lose money on delivery
  - But restaurants can't avoid delivery—because traffic is controlled by platforms

3. **Data Lock-in (Switching Costs):**

• **Salesforce's Data Cage:**
  - Enterprises stored 10 years of customer data in Salesforce
  - Switch to other CRM? Data migration cost = $5-50M
  - Even if competitors are free, enterprises don't dare switch
  - Value capture mechanism: Data hostage

• **WeChat's Social Data:**
  - 1.3B users, 10 years of chat history, Moments, payment records
  - Switch to other social apps? Your social graph can't migrate
  - This is why ByteDance, Alibaba's multiple social attempts failed—data moat too deep

4. **Brand Premium (Cognitive Monopoly):**

• **Hermès's Scarcity Pricing:**
  - Birkin bag manufacturing cost: ~$1,500
  - Official price: $8,000-15,000
  - Secondary market price: $15,000-30,000 (certain styles)
  - Value capture rate: 85-90% (cost only 10-15% of price)
  - Mechanism: Artificial scarcity (limited production + quota system) + brand narrative (craftsmanship + heritage)

• **Moutai's Cultural Binding:**
  - Production cost: ~¥200/bottle
  - Factory price: ¥1,499/bottle
  - Market price: ¥2,500-3,000/bottle
  - Value capture rate: 92% (gross margin)
  - Mechanism: Binding to Chinese business culture ("must drink Moutai for business deals") + scarcity (limited production)

5. **Time Arbitrage (First-mover Advantage):**

• **Coca-Cola's Century-long Mind Share:**
  - 130 years of brand accumulation
  - "Cola" category = Coca-Cola (category is brand)
  - Even if Pepsi tastes better (blind test winner), market share still far below Coke
  - Value capture mechanism: Mind occupation (owning "authentic cola" perception)

• **Tencent's Time Compounding:**
  - QQ (1999) → WeChat (2011) → WeChat Pay (2013) → Mini Programs (2017)
  - Each step built on previous user base
  - Latecomers, even with better products, can't replicate 20 years of time accumulation

**Value Capture Counterexamples (Create Value But Can't Capture):**

• **Open Source Software:**
  - Linux created trillions in value for world (servers, Android foundation)
  - Linux Foundation annual revenue: $170M
  - Value capture rate <0.01%
  - But Red Hat (provides Linux enterprise services) acquired by IBM for $34B—key to value capture is "service" not "software"

• **Airline Industry:**
  - 1903-2023, aviation created immeasurable value for humanity (shortened global distances)
  - But aviation industry net cumulative loss (all airlines' historical profits sum <0)
  - Who captured the value? Passengers (low fares), Boeing/Airbus (sell planes), airports (landing fees), oil companies (fuel)
  - Airlines: Create value, but weakest bargaining power in value chain

• **Scientists:**
  - Einstein's relativity created infinite value for world (GPS, nuclear energy, quantum physics)
  - Einstein's wealth: Middle-class level
  - Compare: iPhone uses GPS technology, Apple market cap $2.8T
  - This is the brutal gap between "creating value" and "capturing value"

**How to Improve Value Capture Ability:**

1. **Establish Pricing Power:**
   - Through brand, patents, network effects, monopoly position
   - Case: Nike doesn't manufacture shoes (outsourced), but captures 70% of profits through brand

2. **Control Critical Nodes:**
   - Occupy "irreplaceable" position in value chain
   - Case: Visa/Mastercard don't issue cards or lend money, but control payment network, capture 51% net margin

3. **Design Lock-in Mechanisms:**
   - Data lock-in, ecosystem lock-in, habit lock-in
   - Case: Apple through ecosystem (iPhone + Mac + iPad + Watch + AirPods) locks customers for life

4. **Continuous Value Capture Models:**
   - From "one-time transactions" to "subscription/membership/consumables"
   - Case: Adobe from selling software (one-time $699) to subscription ($636/year × lifetime)

5. **Cognitive Monopoly:**
   - Occupy "category first" position in customer minds
   - Case: Mention search = Google, e-commerce = Amazon, social = Facebook

**Actionable Steps:**

1. **If you're an entrepreneur:**
   - When designing products, simultaneously design "value capture mechanisms"
   - Don't be the party "creating value but getting commission extracted," be the "commission-extracting platform"

2. **If you're an investor:**
   - Don't just look at "how much value created," more importantly "how much value captured"
   - Prioritize high value capture rate companies: SaaS, platforms, brands, patents

3. **If you're a professional:**
   - Don't just "create value" (work hard), learn to "capture value" (promotions, raises)
   - Build personal brand, control critical resources, become irreplaceable

Warren Buffett's ultimate wisdom: "I'd rather have a mediocre business with excellent value capture mechanisms than a great business with poor value capture mechanisms."

Does this overturn your understanding of "value creation"?
